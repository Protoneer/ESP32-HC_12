## Hardware
- __M5 Stamp C3 doco__ - https://docs.m5stack.com/en/core/stamp_c3

## Wiring 
- ESP32 Pin 6 = radio rx
- ESP32 Pin 7 = radio tx
- ESP32 Pin 8 = Radio Set
- ESP32 Gnd = Radio GND
- ESP32 3V3 = Radio VCC
